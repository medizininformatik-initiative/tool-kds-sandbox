# tool-kds-sandbox

## Kontext:
Hier entsteht im Rahmen des Aus-, Fort- und Weiterbildungsprojekts baseTrace eine Kerndatensatz-Schulungsumgebung (CDS-Schulungsumgebung).

## Zielgruppe:
* Neue Mitarbeitende der DIZe

## Themen:
* Implementierung von CDS-Profilen
* Verarbeiten von FHIR-Beispieldaten
* Validierung von Profilen

> Die Schulungsumgebung soll Mitte 2026 mit einem initialen Satz an Übungen/Inhalten verfügbar werden.
> Zukünftig können weitere Übungen/Inhalte entwickelt und hinzugefügt werden.
